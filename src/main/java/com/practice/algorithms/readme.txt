1、欧几里得算法:gcd.java
    计算两个非负整数p和q的最大公约数：
    若q是0，则最大公约数是p。
    否则，将p处以q得到余数r,p和q的最大公约数即为q和r的最大公约数

2、读取算术表达式并计算  ExpressionCalculator.java
    原理：2个栈来实现，一个保存操作数，一个保存操作符
          读取到数字的时候，将数字压入操作数栈，
          输入到符号的时候，将符号压入操作符栈，
          其中读取到“）”时，分别从操作数栈和操作符栈中弹出数字和操作符操作，结果在压入操作数栈

3、可以动态调整数组大小的实现 栈  ResizingArrayStack.java
    动态调整数组的大小：将栈移动到另一个大小不同的数组中
    当数组满了，则动态将数组长度加倍
    当栈大小小于数组的四分之一的时候，将数组长度减半
    实现一个迭代器，逆序迭代遍历这个数组
